  1. canvas 绘制 10 个圆
    canvas 绘制 100 个圆
<canvas id='canvas'  height="1000" width="1000"></canvas>
  <script>
    var canvas=document.getElementById('canvas');
    var ctx=canvas.getContext('2d');

    for(var i=1;i<=10;i++) {
      ctx.beginPath();
      ctx.arc(10*i,10*i,5,0,Math.PI*2,true)
      ctx.stroke()
    }
  </script>

小知识：

  • 在 canvas 标签设置 width,height 是设置的绘图表面和元素本身的大小

  • 在 css 里设置 canvas 的 width,heigth 是设置的元素本身的大小

  • 由于上面两个原因，当绘图表面比元素本身小时就会产生拉伸效果，这儿是例子：demo

  1. HTTP 502
    网关错误。
    此服务器访问一个上游服务器， 但收到无效响应。这通常并不意味着上游服务器已关闭（对网关 / 代理无响应）， 而是上游服务器和网关 / 代理在交换数据的协议上不一致。

以下是我整理的面试题，按技能点区分

HTML

  1. （必考） 你是如何理解 HTML 语义化的？
    概念：用有意义的标签来编码
    历史：tabel 布局—-div+css 布局—-h1 p article 专业的语义化标签
    好处：便于阅读维护 利于 SEO 利于其他设备的解析（盲人阅读器）

  2. <strong> 和 <b> 和 <em> 和 <i> 的区别？
    <strong> <b> 都是字体加粗，但是前者表示强调重点内容，后者没有实际含义
    <em> <i> 都是字体变斜体，但是前者表示强调，引用等，后者没有实际含义
    我们尽量使用<strong> <em> 这两个有语义化的标签

  3. 块级元素和行内元素区别？
    块级元素：

    1. 独占一行 2. 可设置宽高

    行内元素：

    1. 不独占一行 2. 宽高由内容物决定

  4. doctype 是什么？它的作用？
    声明文档类型，作用是：告诉浏览器以何种文档类型规范来解析文档

  5. meta viewport 是做什么用的，怎么写？
    做移动端适配用的，使用它来自适应设备屏幕宽度

<meta name="viewport" content="width=device-width,user-scalabel=no" />
  • width=device-width 宽度设置为理想视口宽度
  • user-scalabel=no 不允许用户去放大缩小
  1. 你用过哪些 HTML 5 新增的标签？
  • <audio></audio> 音频
  • <video></video> 视频
  • <canvas></canvas>
  • <header></header>
  • <footer></footer>
  • <nav></nav>
  • <section></section> 定义文章中的区段
  • <article></article> 论坛，评论等
  • <main></main> 定义文档主要内容
  • <aside></aside> 内容可以用作文章的侧边栏

  • autoplay

    插入视频

  1. 两种盒模型
  • 标准盒子模型：width=content
  • IE 盒子模型: width=content+padding+border
    可用 css3 的 box-sizing 属性切换盒子模型:box-sizing:content-box/border-box

  1. CSS 选择器优先级
    !important > 行内样式>ID 选择器 > 类选择器=伪类=属性 > 标签 > 通配符 > 继承 > 浏览器默认属性

  2. css3 里的动画是怎样写的？
    展现动画效果
    属性有：animation-name animation-duration animation

    .demo{
width:100px
animation: name 2 infinite
}

@keyframe name{
from{width:100px}
to{width:200px}
}

  3. css3 里的过渡是怎样写的？
    属性有:transition-property transition-duration transition

    .demo {
  width: 200px;
  transition: width 3s;
}
.demo:hover {
  width: 300px;
}

  4. css3 中的媒体查询？
    根据不同的媒体类型来定义不同的样式

    @media screen and (max-width: 300px) {
  body {
    width: 200px;
  }
}

  3. let 和 var 和 const 区别，具体用 var 会出现什么错误

    1. （块级作用域） let 可以声明块级作用域的变量，而 var 不行
    2. （变量提升） let 不存在变量提升，只允许在 let 声明之后才能使用变量，而 var 存在变量提升
    3. （变量重复声明）let 不能重复声明一个变量，var 可以
    4. const 声明的是常量，不能修改，也就是不能修改指向那个值的地址
for (var i = 0; i < 10; i++) {
  setTimeOut(i => {
    console.log(i)
  }, 1000)
}

Vue

  1. vue-router 是什么？
    vue-router 就是路由，地址栏输什么，就显示什么页面，就是页面跳转。它还可以实现页面间的传参

  2. vuex 是什么？
    状态管理工具，就是集中管理所有组件需要共用的属性

  3. axios 是什么，有哪些常用方法？
    是一个基于 Promise 的一个库，发送请求时用
    方法：

    • axios.get
    • axios.post
    • axios.put
    • axios.delete

  4. git 常用命令？代码仓库协同工作流程？

    • git init 初始化仓库，变成我们可以管理的仓库
    • git add 文件添加到暂存区
    • git commit -m ‘说明’ 暂存区文件提交到分支（默认 master）
    • git status 查看工作区和暂存区状态
    • git push 提交到远程仓库
    • git clone 克隆远程仓库
    • git checkout 切换分支

    工作流程：
    fork 别人的仓库，相当于建立了一个分支
    然后开发者 B 在 clone 自己的 Github 上的仓库到本地
    fetch 从远程仓库中抓取本地仓库中的没有的数据
    在 Github 上 Pull request 一下，就是给主开发者 A 发送一个请求，请求合并项目
    merge 合并一下分支

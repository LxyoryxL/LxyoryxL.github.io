软件安装和快捷键

常识： mac 多一个 cmd 键，学习盲打工作效率高，卸载 360

软件安装

  • 安装 SwitchyOmega 一个代理切换工具
  • 安装 uBlock 关闭广告
  • 安装 git easy
  • 安装 cmder 它比 git bash 还要强大 s
  • 安装 nrm
  • 安装 yarn 在官网下载，不要用 npm 下载

win10 常用快捷键：

win+ctrl+方向键 切换桌面
ctrl+Y 重做
ctrl+P 打印
鼠标中键 关闭页面
ctrl+D 创建新页面
ctrl+L 到地址栏输入
alt+左/右 history 前进后退

网页控制台的秘密：

网页控制台->source->snippest 可以保存代码片段
网页控制台->Network->模拟 3G 做调试用
网页控制台->Network->preserve log
网页控制台->Network->Disable cache 清空缓存
理解 preserve log
理解 Disable cache

vscode 操作：

直接在命令行输入`code` 可以打开 vscode
ctrl+H 替换
ctrl+N 新建文件
ctrl+P 查找文件
ctrl+shift+P 各种设置
按住 alt 多光标选择

cmder 操作：

ctrl+T 新建一个 cmd
ctrl+f 弹出 cmd
ctrl+w 关闭 cmd
alt+d 建立分屏
ctrl+d 删除分屏
鼠标选中就可以复制
鼠标右键直接粘贴
也可以使用经典的 cv 来复制粘贴

命令行常用单词

make move remove list recursive link find echo touch directory force copy

程序员吧动词缩写，就变成了命令：

  • make==>mk
  • remove==>rm
  • list==>ls
  • copy==>cp

文件的增删改查（命令行）

增:创建文件

- 创建 1.txt
   touch 1.txt
   touch 1.txt 2.txt    （同时创建多个文件）
   echo hi > 1.txt      （> 表明覆盖原来文件内容）
   echo hi >> 1.txt     （>> 表明追加到原来文件内容里面）
- 创建目录 a
   mkdir a
   mkdir -p a/b/c
   mkdir -p a/b/c a/b/m （同时创建多个目录）
- 复制文件 1-->2
   cp 1.txt 2.txt       （复制1.txt的内容 到 2.txt里）
- 复制目录
   cp -r a b            （复制文件a 到 文件b中）

删

- 删除 1.txt
  rm 1.txt              （注意：用该命令行删除的文件是在回收站找不到的了，所以慎用）
- 删除目录 a/
  rm -r a/
- 强制删除 a
  rm -rf a

注意：c 盘里的 文件大多都不能删,除了‘用户’目录，缩写为~
一般做操作时都在~下的目录

改

- 打开文件  1.txt
   start 1.txt
- 清空文件
   echo '' > 1.txt
- 修改文件内容
   echo hi > 1.txt      （> 表明覆盖原来文件内容）
- 追加文件内容
   echo hi >> 1.txt     （>> 表明追加到原来文件内容里面）
- 移动文件/目录
  mv
- 重命名文件/目录
  mv
- 修改文件最后更新时间
  touch 1.txt           （如果 1.txt 存在，就修改 1.txt 的最后更新时间）

查

- 查看当前目录的绝对路径
  pwd
- 查看当前目录下的所有文件
  ls
- 查看指定目录下的文件
  ls 路径   例如：
                  - ls d/blogg
                  - ls /d/blogg 前面加 / 为在根目录下查找
- 查看文件内容
  1. cat [文件名]
     例如：cat 1.txt
  2. head -n [数量][文件名]
     例如：head -n 20 1.txt
  3. tail -n [数量][文件名]
     例如： tail -n 20 1.txt
  4. less [文件名]
     例如：less 1.txt

如此多的命令怎么才能记得住呢？查啊！
但是通过 cp --help 提供 的文档真是 too long didn’t read…..
于是有了另一种简单的查文档插件 tldr:
装插件：tldr
使用命令 例如 tldr mkdir 来查看命令意思

命令的组合

&& 操作
一条命令成功之后，执行另一条
例如：
mkdir c && echo ‘创建了 c 目录’

;操作
不管成功失败，都执行下一条
例如：
mkdir d ; echo ‘创建了 d 目录’
